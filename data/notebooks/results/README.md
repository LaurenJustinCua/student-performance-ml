# An Analysis of Student Academic Performance Using Machine Learning

## Module
Machine Learning

## Programme
BSc (Hons) Creative Computing

## Description
This project applies supervised machine learning techniques to predict whether a student will pass or fail based on academic, demographic, and social attributes. Logistic Regression and Random Forest models were implemented and evaluated using accuracy, classification reports, and confusion matrices.

## Dataset
Student Performance Dataset sourced from Kaggle (UCI Machine Learning Repository).

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
